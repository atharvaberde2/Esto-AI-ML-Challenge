# Esto AI/ML Challenge

This repository contains code for the Esto AI/ML Challenge, a machine learning project.

## Requirements

The following Python packages are required:

- pandas
- numpy
- scikit-learn
- matplotlib

## Installation

Install the required packages using pip:

```bash
pip install -r requirements.txt
```

## Usage

### Running the Training Script

To run the training script locally:

```bash
python train.py
```

### Using in a Jupyter Notebook

If running in a Jupyter notebook environment:

1. Install the requirements:

   ```python
   !pip install -r requirements.txt
   ```
